# SwiperEats

SwiperEats is a Tinder-like mobile application designed to help indecisive diners choose where to eat. Users can browse restaurants by swiping right to add to their shortlist and left to skip.

## How to Run

### Prerequisites

- Node.js (v18 or newer)
- npm
- Expo Go app on your mobile device (optional, for testing on physical devices)

### Setup and Execution

1. **Navigate to the Frontend directory:**
   ```bash
   cd Frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run start
   ```

4. **Testing the app:**
   - **Android:** Press `a` in the terminal to open on an Android emulator or device.
   - **iOS:** Press `i` in the terminal to open on an iOS simulator.
   - **Web:** Press `w` in the terminal to open in your web browser.
   - **Mobile Device:** Scan the QR code shown in the terminal using the Expo Go app.
